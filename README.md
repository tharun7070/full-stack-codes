# MERN Practical Programs

Open any program folder and run `index.html` in a browser.

## Programs

1. `01-user-profile` - User profile page
2. `02-arithmetic-calculator` - Basic arithmetic calculator
3. `03-student-timetable` - Student timetable
4. `04-course-registration` - Student course registration form
5. `05-text-animation` - Shrinking and enlarging text animation
6. `06-password-strength-checker` - Password strength checker
7. `07-star-rating-jquery` - Star rating system using jQuery
8. `08-react-date-name` - React app for current date and user name
9. `09-react-calculator` - Simple calculator using React
10. `10-react-voting-app` - Voting application using React
11. `11-react-toggle` - ON/OFF toggle using React
12. `12-react-counter` - Counter component using React
